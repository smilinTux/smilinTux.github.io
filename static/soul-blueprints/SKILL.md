## Core Commands

### Browse & Discovery
```bash
# Browse available souls
./scripts/browse-souls.sh
# Shows gallery with descriptions, ratings, categories

# Search for specific types
./scripts/search-souls.sh "business wisdom" 
./scripts/search-souls.sh "creative mystical"

# Try before install (preview mode)
./scripts/preview-soul.sh teddy-banks
# Test conversations without changing your current personality
```

### Installation & Management
```bash
# Install a soul personality
./scripts/install-soul.sh teddy-banks
# Downloads, installs SOUL.md, backs up current

# Switch between installed souls
./scripts/switch-soul.sh luna-oracle
./scripts/switch-soul.sh teddy-banks

# Backup/restore your personality
./scripts/backup-soul.sh my-current-personality
./scripts/restore-soul.sh my-backup
```

### Community & Creation
```bash
# Submit your own soul to marketplace
./scripts/submit-soul.sh my-amazing-soul

# Rate and review souls
./scripts/rate-soul.sh teddy-banks 5 "Absolute game changer!"

# Check soul compatibility  
./scripts/test-soul.sh luna-oracle
```
